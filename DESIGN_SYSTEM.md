# **🎨 UI-концепт: Solo Leveling (The Monarch)**

### 1️⃣ Основной экран (Dashboard)

* **Фон:** Глубокий синий `#0B1D51` + subtle digital noise.
* **Главный заголовок:** Orbitron, неоновый фиолетовый `#9B59B6`, subtle glow.
* **Карточки квестов:**

  * Sharp-edged панели с inner glow.
  * Hover: glitch animation + slight shake.
  * Активные квесты: красный `#E74C3C` border pulse.
* **HUD (HP, EXP, Mana):**

  * Neon bars, subtle flicker.
  * При изменении: плавная анимация fill + glitch spark.
* **Иконки:**

  * Boss, Loot, Quest — пиксель-арт / техно-стиль.
* **Частицы:** floating sparks, цифровые полосы, падающий ASCII-код.
